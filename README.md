# Price Watch API

API para monitoramento de preços de produtos.

Este projeto será desenvolvido em Python com FastAPI, SQLAlchemy, SQLite e BeautifulSoup.

## Status

Em desenvolvimento.

## Funcionalidades planejadas

- Cadastrar produtos para monitoramento
- Coletar nome, preço e disponibilidade via scraping
- Salvar histórico de preços
- Consultar histórico via API
- Enviar alertas quando o preço cair
